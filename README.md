# gfd-trial

This is a short-term experiment to see if
[autodb](https://github.com/CharnelMouse/autodb) can be extended to
automatically discover and synthesise null-free schemas for a given dataset.

Unfortunately, asking quarto to render the results in Markdown causes some
glaring issues with the Graphviz images, so seeing the results requires running
the project and opening the resulting HTML file.

This report can be generated from scratch by downloading the project, and
running `targets::tar_make()` in R from the project folder. A list of used
package versions is included at the end of the report, under Session Info.
